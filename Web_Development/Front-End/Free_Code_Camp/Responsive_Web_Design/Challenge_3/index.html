JavaScript: Uma Introdução

JavaScript é uma linguagem de programação de alto nível, leve e interpretada, utilizada principalmente para desenvolvimento web. Foi originalmente desenvolvida por Brendan Eich na Netscape em 1995, com o nome de "LiveScript", mas logo foi renomeada para JavaScript.

Características Principais:

  • Linguagem de Script: JavaScript é uma linguagem de script, o que significa que os comandos são executados linha por linha, sem a necessidade de compilação.
  • Multiplataforma: Pode ser executada em diferentes plataformas, incluindo navegadores web, servidores, dispositivos móveis, entre outros.
  • Interpretação: O código JavaScript é executado diretamente no navegador do cliente, o que o torna mais flexível e dinâmico.
  • Client-Side e Server-Side: Pode ser usado tanto no lado do cliente (navegador) quanto no lado do servidor (Node.js).

Utilização Principal:

  • Desenvolvimento Web: JavaScript é amplamente utilizado para criar interatividade em páginas da web, incluindo animações, validação de formulários, efeitos visuais dinâmicos, entre outros.
  • Desenvolvimento de Aplicativos Web: Frameworks como React, Angular e Vue.js permitem o desenvolvimento de aplicativos web robustos e escaláveis.
  • Desenvolvimento de Jogos: Com bibliotecas como Phaser e Three.js, é possível criar jogos interativos diretamente no navegador.
  • Desenvolvimento de Aplicativos Móveis: Frameworks como React Native e Ionic permitem o desenvolvimento de aplicativos móveis usando JavaScript.
Variáveis e Tipos de Dados

JavaScript é uma linguagem de programação de tipagem dinâmica, o que significa que você não precisa especificar explicitamente o tipo de uma variável.
Exemplo de Declaração de Variáveis:

// Declaração de variáveis let nome = "João"; const idade = 25; var altura = 1.75; // Tipos de dados let texto = "Isso é uma string"; let numero = 42; let booleano = true; let array = [1, 2, 3]; let objeto = { nome: "Maria", idade: 30 };
Estruturas de Controle de Fluxo

JavaScript oferece várias estruturas de controle de fluxo para executar diferentes blocos de código com base em condições.
Exemplo de Estruturas de Controle de Fluxo:

// If-else let hora = 12; if (hora < 12) { console.log("Bom dia!"); } else if (hora < 18) { console.log("Boa tarde!"); } else { console.log("Boa noite!"); } // Loop while let contador = 0; while (contador < 5) { console.log(contador); contador++; } // Loop for for (let i = 0; i < 5; i++) { console.log(i); }
Funções

As funções permitem reutilizar blocos de código em seu programa.
Exemplo de Declaração de Função:

// Declaração de função function saudacao(nome) { return "Olá, " + nome + "!"; } // Chamada de função console.log(saudacao("Ana")); // Saída: Olá, Ana!
Eventos

JavaScript é frequentemente usado para interagir com elementos HTML, respondendo a eventos do usuário.
Exemplo de Tratamento de Eventos:

// Adicionando um evento de clique ao botão document.getElementById("meuBotao").addEventListener("click", function() { alert("Botão clicado!"); });
Manipulação do DOM

JavaScript pode ser usado para manipular elementos HTML e CSS no Document Object Model (DOM).
Exemplo de Manipulação do DOM:

function alterarTexto() { document.getElementById("meuDiv").innerText = "Novo texto!"; }
